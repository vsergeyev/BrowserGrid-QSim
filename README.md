# A Feasibility Study of Browser-Based Distributed State-Vector Quantum Circuit Simulation

This repository contains the artifacts and code for the research on browser-based distributed State Vector
simulator for Quantum Circuits. A state vector simulator is a powerful utility that models quantum circuits
by calculating the EXACT mathematical state (amplitudes and probabilities) of the qubits. Such approach is
considered the "gold standard" for quantum algorithms development:

* State Vector simulator delivers exact, noiseless results.
* State Vector simulator allows deep internal inspection of quantum states.
* State Vector simulator is a classical tool and not require actual Quantum hardware to run.
